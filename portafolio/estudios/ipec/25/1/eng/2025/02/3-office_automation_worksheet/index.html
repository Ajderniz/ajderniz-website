7. What is a Bit?

It is the smallest amount of data possible. It represents a state of 0/1, ON/OFF, YES/NO.

8. What is a Byte?

In modern personal computers, a byte is a set of 8 adjacent bits.

Throughout the history of computing this number has shifted slightly, and there are still some devices which utilize distinct amounts of bits.

If you wanna get technical, you can define a byte as the smallest amount of addressable data possible.

9. What is the difference between 32 bits and 64 bits computer>

I kinda forgot about this, so I had to look it up. Funnily enough, its kinda related to the previous question. So, what 32-bit and 64-bit refers to is the amount of bits a memory (RAM) address contains.

Let us remeber what I said on the previous answer, that a byte is the least amount of addressable data a modern PC's processor can point to. Each byte is represented by one of these 32-bit addresses; so, in essence, a 32-bit computer is capable of having 32-bits worth of RAM, while a 64-bit computer is capable of 64-bits worth of RAM.

I'd do the math, but I'm dumb, so I'll copy what came up on the search query: with 32-bits, you can access up to 4GB, while with 64-bits, you can access, theoretically, up to 18.4 million terabytes, which is, by current standards, a lot.
