Some IoT Devices

2025-04-01 by Ajderniz, tagged as iot

Foreword

So, here's a list of some IoT devices suggested by the students today in class, and some characteristics that make them so cool and different from normal, non-broke, devices without internet access. Enjoy.

Main list

Smart fridge

  • Remote control and configuration
  • Inventory management
  • Recipe suggestions

Smart lights

  • Voice-activation
  • Remote control via voice or smartphone apps
  • Color and/or brightness adjustment

Smartwatch

  • Touchscreen
  • Can run mobile apps
  • Essentially, a portable computer or smartphone

Smart microwave

  • Control via voice commands
  • Can be connected to a mobile app
  • Features like auto-defrost, and some presets

Smart washing machine

  • Allows for greater customization
  • Can be controlled remotely
  • Features energy efficiency control

Smart security alarm

  • Can be integrated into a smart security system
  • Can be accessed... (guess)... remotely
  • Motion detection and other sensors whose activation may be monitored through an app or whatever

Alexa

  • Has no "dumb" counterpart
  • Can set up alarms and perform some other tasks
  • Voice-controled
  • Can be used to control other devices

Siri Assitant

  • Implemented into Apple platforms
  • Controled via voice and text prompts
  • Performs searches and answers questions, among other functions

Smart earbuds (e.g. Galaxy Buds)

  • Remotely controlled
  • Features GPS location via an account ID
  • Can be monitored and charged by a linked smartphone

Smart printer

  • Connected to the internet
  • Can be prompted to print documents wirelessly
  • May include features like scanning and faxing (via the interwebbbs)
  • Gets updates that help the manufacturer take advantage of the owner

Tablet

  • Larger than a phone
  • Can be used to read books and CONSUME other media
  • Able to run applications

Smart TV

  • Connects to the internet, as usual.
  • Can be used for more than watching TV, like browsing the interwebbbbb, streaming videos and so on.
  • Essentially a smart phone embedded into a TV

Smart mirror

  • In contrast to normal mirrors, this is an abomination
  • Apart from a reflection, shows information such as the weather, time and notifications
  • May be able to be integrated into a smart home environment

Smart egg tray (Egg Minder)

  • Keeps track of eggs
  • Monitors eggs
  • Makes you think about eggs
  • Eggs

Smart condom (i.Con)

  • It's connected to the network
  • It's wrapped around your micropenis
  • Vibrates or something, I don't know honestly

Smart (not really) juice box puncturer and extractor (Juicero)

Smart flip-flops
